# Customer Segmentation Project

## Overview
This project applies K-Means clustering to segment customers based on their 
Age, Annual Income, and Spending Score. The goal is to identify distinct 
customer groups (e.g., high-income low-spenders, young high-spenders) to 
support targeted marketing strategies.

## Dataset
Sample customer dataset with 20 records containing:
- CustomerID
- Gender
- Age
- Annual Income (in thousands)
- Spending Score (1-100)

## Tools Used
- Python (Google Colab)
- pandas, numpy
- scikit-learn (K-Means Clustering)
- matplotlib / seaborn (visualization)

## Approach
1. Data exploration and cleaning
2. Feature selection (Age, Income, Spending Score)
3. K-Means clustering to group customers into segments
4. Visualization of clusters
5. Business insights based on segment characteristics
